The programmatic API lets you run imports without the Livewire wizard. Use it in artisan commands, API endpoints, scheduled tasks, or anywhere you need headless CSV processing.

::note
The programmatic API is part of `tapix/core` and requires no UI dependencies. Install with `composer require tapix/core`.
::

## ImportManager

`ImportManager` provides the full import pipeline as method calls:

### Creating an Import

```php
use Tapix\Core\Tapix;
use App\Importers\ContactImporter;

$manager = Tapix::import(ContactImporter::class);
$import = $manager->createFromFile('/path/to/contacts.csv');
```

`createFromFile()` parses the file, creates the Import model, and loads all rows into the per-import SQLite store. Returns the `Import` model.

Optional `userId` parameter to set the import owner:

```php
$import = $manager->createFromFile('/path/to/contacts.csv', userId: $user->id);
```

### Auto-Mapping Columns

```php
$mappings = $manager->autoMap($import);
```

Automatically matches CSV headers to importer fields using name matching and data type inference. Saves the mappings to the Import model and returns the mapping collection.

### Custom Mapping

```php
use Tapix\Core\Data\ColumnData;

$mappings = collect([
    ColumnData::toField('Full Name', 'name'),
    ColumnData::toField('Email Address', 'email'),
    ColumnData::toField('Phone', 'phone'),
]);

$manager->applyMapping($import, $mappings);
```

### Validation

```php
$batchIds = $manager->validate($import);
```

Dispatches async validation jobs for each mapped column and a match resolution job. Returns batch IDs keyed by column source name plus `__match_resolution`.

### Executing the Import

```php
$batchId = $manager->execute($import);
```

Dispatches the `ExecuteImportJob` and returns the batch ID. The import processes asynchronously via the queue.

## Example: Artisan Command

```php
use Illuminate\Console\Command;
use Tapix\Core\Tapix;
use App\Importers\ContactImporter;

class ImportContactsCommand extends Command
{
    protected $signature = 'contacts:import {file}';
    protected $description = 'Import contacts from a CSV file';

    public function handle(): void
    {
        $manager = Tapix::import(ContactImporter::class);

        $this->info('Parsing file...');
        $import = $manager->createFromFile($this->argument('file'));
        $this->info("Found {$import->total_rows} rows.");

        $this->info('Auto-mapping columns...');
        $manager->autoMap($import);

        $this->info('Dispatching import...');
        $manager->execute($import);

        $this->info("Import {$import->id} dispatched to queue.");
    }
}
```

## Example: API Endpoint

```php
use Illuminate\Http\Request;
use Tapix\Core\Tapix;
use App\Importers\ContactImporter;

Route::post('/api/imports', function (Request $request) {
    $request->validate(['file' => 'required|file|mimes:csv,xlsx']);

    $manager = Tapix::import(ContactImporter::class);
    $import = $manager->createFromFile(
        $request->file('file')->getRealPath(),
        userId: auth()->id(),
    );
    $manager->autoMap($import);
    $batchId = $manager->execute($import);

    return response()->json([
        'import_id' => $import->id,
        'batch_id' => $batchId,
        'total_rows' => $import->total_rows,
    ]);
});
```

## Monitoring Progress

After dispatching, poll the Import model for status:

```php
use Tapix\Core\Models\Import;
use Tapix\Core\Enums\ImportStatus;

$import = Import::find($importId);

if ($import->status === ImportStatus::Completed) {
    echo "Created: {$import->created_rows}, Updated: {$import->updated_rows}";
}
```
